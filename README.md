# Time-Frequency Detection

This repository provides supplementary materials for reproducing the YOLO11n-CT-based time-frequency detection experiments.

## Contents

- `Model/`: trained PyTorch and ONNX model weights.
- `Samples/`: representative anonymized spectrogram samples and corresponding YOLO-format labels.
- `Source Code and Config Files/`: model configuration files, training scripts, ONNX inference scripts, and the TFCM implementation.

## Data Availability

The original acoustic recordings are not publicly released due to project and data-use restrictions. To improve reproducibility, representative spectrogram samples and corresponding YOLO-format labels are provided in `Samples/`.

## Code Usage

The scripts were used in the original experimental environment and may contain absolute paths. Before running the code, users should modify the dataset paths, model paths, and output paths according to their local environment.

The provided materials are intended to support academic research and reproducibility of the reported experiments.
